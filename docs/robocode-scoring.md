# Robocode 经典版 · 记分规则

> 来源：[RoboWiki / Scoring](https://robowiki.net/wiki/Robocode/Scoring)、[The Book of Robocode 速查表](https://book.robocode.dev/appendices/quick-reference.html)。
>
> **核心观念**：在 Robocode 里，**活到最后 ≠ 赢**。最终排名看的是**总分（Total Score）**，而总分主要由“造成多少伤害”驱动。一个躲得很好但不开火的机器人，分数往往低于一个被打死、但打出大量伤害的机器人。所以 MechGarden 的机器人在设计时要兼顾**生存**与**输出**。

## 总分构成

最终排名 = 各项分数之和（**Total Score**）。各项如下：

| 分项 | 规则 | 公式 |
|------|------|------|
| **存活分** Survival | 每当**别的**机器人死亡，所有当时还活着的机器人各得 50 分 | `+50 × (在你存活期间死掉的对手数)` |
| **最后幸存奖励** Last Survivor Bonus | 一局中最后存活的机器人，对在它之前死掉的每个机器人额外得 10 分 | `+10 × (在你之前死掉的对手数)` |
| **子弹伤害分** Bullet Damage | 对敌人造成的每 1 点子弹伤害得 1 分 | `+1 × 子弹伤害总量` |
| **子弹击杀奖励** Bullet Damage Bonus | 用子弹**击杀**某敌人时，额外获得你对**该敌人**累计子弹伤害的 20% | `+0.20 × 你对该敌人的子弹伤害累计` |
| **撞击伤害分** Ram Damage | 通过撞击对敌人造成的每 1 点伤害得 2 分（撞击得分是子弹的 2 倍） | `+2 × 撞击伤害总量` |
| **撞击击杀奖励** Ram Damage Bonus | 通过撞击**击杀**某敌人时，额外获得你对**该敌人**累计撞击伤害的 30% | `+0.30 × 你对该敌人的撞击伤害累计` |

## 各分项详解

### 存活分（Survival Score）
- **被动获得**：你不需要做任何事，只要在对手死亡的那一刻你还活着，就 +50。
- 这是“生存”的直接回报，鼓励机器人活得久。
- 1v1 中，赢一局（对手先死）= +50 存活分。

### 最后幸存奖励（Last Survivor Bonus）
- 只发给一局里**最后活着**的那一个机器人。
- 1v1 中只有 1 个对手，所以最后幸存奖励 = +10。

### 子弹伤害分（Bullet Damage）
- **得分主力**。造成多少伤害就得多少分（1:1）。
- 单发子弹伤害见物理文档：`4×power`（power>1 时再加 `2×(power−1)`）。
- 这是为什么“能打中”比“能躲”更重要——命中率直接决定排名。

### 子弹击杀奖励（Bullet Damage Bonus）
- 把敌人**用子弹打死**时，按你对**这个敌人**全程累计的子弹伤害再奖励 20%。
- 鼓励“持续锁定同一个目标直到击杀”，而不是雨露均沾。

### 撞击伤害分（Ram Damage）& 撞击击杀奖励
- 撞击得分单价是子弹的 2 倍（2 分/点伤害），击杀奖励比例也更高（30%），但**撞击同时让自己也掉血**（双方各 0.6/次，见物理文档），且需要贴脸，风险高。
- 一般作为 melee 或近身侵略流（如规划中的 *Mantis*）的辅助手段，1v1 基线机器人通常以子弹为主。

## 结果表的列含义

对战结束后 Robocode 弹出的结果表，各列对应上面的分项：

| 列名 | 含义 |
|------|------|
| **Total Score** | 总分，决定排名 |
| **Survival** | 存活分（50/次） |
| **Surv Bonus** / Last Survivor | 最后幸存奖励（10/对手） |
| **Bullet Dmg** | 子弹伤害分 |
| **Bullet Bonus** | 子弹击杀奖励 |
| **Ram Dmg** | 撞击伤害分 |
| **Ram Bonus** | 撞击击杀奖励 |
| **1sts / 2nds / 3rds** | 各名次出现次数（仅统计，不计入总分） |

## 对 MechGarden 机器人设计的启示

1. **输出 > 苟活**：单纯躲弹不开火排名会很低，必须保证命中输出。
2. **命中率是排名核心**：子弹伤害分 + 击杀奖励几乎都来自“打中”。提升瞄准（线性 → 圆周 → GuessFactor）是涨分最快的方向。
3. **集中火力**：击杀奖励按“对单一敌人的累计伤害”计算，鼓励锁定单一目标打到死（1v1 天然满足，melee 要主动选目标）。
4. **撞击是高风险高回报的补充**，非 1v1 主线。
5. **能量经济学**：开火耗能、命中返还 `3×power`，要在“多开火涨伤害分”和“别把自己耗到禁用”之间平衡。
